This website will allow new and/or current users to login/register and upload their assets using longitude and latitude data to add an asset into the database which has all assets stored. To upload a new asset, go to the upload page and either use the file upload button, or manually enter the longitude and latitude data into the text boxes and click submit.

Please note that any files which are uploaded need to be in a CSV file type or the system will not accept the data uploaded.
